# Odoo Bank Hours

This module allows you to manage bank hours within Odoo applications effectively. It is designed to help users keep track of the operational hours of various bank branches.

## Features
- Customize bank operating hours for each branch.
- User-friendly interface for managing hours.
- Integration with other Odoo modules.
- Supports localization with proper UTF-8 encoding.

## Installation
1. Go to the Odoo Apps marketplace.
2. Search for "Odoo Bank Hours".
3. Install the module directly from the interface.

## Usage
Once installed, navigate to the Bank Hours configuration to set the operational hours for each branch.
